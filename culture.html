민족 문화

★ 구한말 ★

1. 교육

● 교육입국조서(갑오개혁)

2. 국어

● 국문연구소 : 국어 문법

● 조선광문회 : 고전정리, 간행

3. 문학

● 신체시 : 해에게서 소년에게

● 신소설 : 금수회의록, 혈의누

4. 연극, 영화

● 원각사(은세계, 치악산)

5. 역사

● 위인전, 외국의 흥망사

● 독사신론(신채호) : 민족주의 사학의 방향 제시

★ 1910년대 ★

1. 교육

● 1차 조선 교육령 : 일제학제 6년 vs 조선학제 4년

2. 문학

● 무정(이광수)

3. 연국, 영화

● 신파극

4. 역사

● 식민사관 : 타율성론, 당파성론, 정체성론

★ 1920년대 ★

1. 교육

● 2차 조선 교육령 : 일제학제 = 조선학제, 조선어 필수

2. 국어

● 조선어 연구회 : 가갸날

3. 문학

● KAPF 문학 : 사회주의 영향

4. 잡지

● 어린이, 한글, 개벽(천도교)

5. 연극, 영화

● 아리랑, 토월회

6. 역사

● 민족주의 사학

● 박은식(혼)

● 신채호(낭가사상, 아.비아 투쟁)

★ 1930년대 & 1940년대 ★

1. 교육

● 3차 조선 교육령 : 조선어 수의과목(선택) -> 사실상 폐지

● 명칭변경 : 보통학교 -> 심상소학교 -> 국민학교(1940년)

2. 국어

● 조선어학회 : 맞춤법 통일안, 우리말 큰사전 편찬시도, 조선어학회사건으로 폐지

3. 문학

● 브나로드 운동 영향(심훈, 상록수)

4. 잡지

● 한글, 진단학보

5. 연극, 영화

● 극예술 연구회

6. 역사

● 조선학 연구(정인보, 얼)

● 실증주의 사학(진단학회)

● 사회경제사학 : 사회주의 영향을 받음, 식민사관의 정체성론 반박

★ 종교 ★

1. 대종료

● 단군 믿음, 중광단

2. 원불교

● 새생활 운동(금연, 금주), 북교의 대중화

3. 천도교

● 제2의 3.1운동 계획, 실천못함.
