1910년대 운동

-일제의 무단통치-

  • ※ 정치 : 결사 및 집회 자유 제한, 보안법, 신문지법, 출판법
  • ※ 교육 : 조선교육령(1911), 서당 규칙(1918)
  • ※ 통치 : 헌병 경찰제, 즉결 처분권, 태형실시(1912)
  • ※ 경제수탈 : 회사령(1910), 산림령(1911), 토지조사사업(1912~1918), 광업령(1915), 임야조사사업(1918)

★ 국내 운동 ★

1. 대한독립의금부(1912)

● 1912년 임병찬이 고종의 밀명을 받고 조직된 독립운동단체

● 복벽주의(나라를 되찾고 군주정을 회복하려는 사상) 주장, 국권반환요구서, 무력항쟁준비

2. 대한광복회(1915)

● 1915년 대구에서 결성된 독립운동단체

● 설립목적 : 국권회복, 독립 달성, 만주에 무관학교 설립 및 독립군 양성

● 주요인물 : 박상진, 김좌진

● 공화정(세습 군주나 선거로 뽑힌 군주 이외의 개인 또는 집단이 통치하는 정치 사상) 주장, 군자금 확보, 무력투쟁준비

3. 신민회(1907)

● 국내에서 결성된 항일 비밀결사단체

● 설립 목적 : 민중계몽, 국권회복, 실력양성

● 대성학교(평양)/오산학교(정주)/보창학교(강화) 설립, 대한매일신보 창간, 민족산업진흥운동 주도, 독립군 양성운동(신흥무관학교)

● 1911년 일제가 조작한 105인 사건을 계기로 해체

4. 기타

● 조선국민회(1915) : 평양 숭실학교 학생들이 조직한 독립운동단체

● 기성단(1914) : 평양에서 조직된 항일 학생조직

● 송죽회(1913) : 평양에서 조직된 여성독립운동단체

● 자립단(1915) : 함경남도 단천엣 조직된 독립운동단체

5. 3.1 운동

● 1919년 3월 1일을 기점으로 일어난 최대규모의 민족 항일독립운동

● 배경 : 민족자결주의(미국 윌슨대통령의 각 민족은 스스로 자국의 정치적 운명을 결정해야한다는 주장), 2.8 독립 선언(일본의 조선유학생들이 독립 선언서 발표)

● 과정 : 3월 1일 33인의 대표자가 구성되어 독립선언문 낭독 → 종로 탑골 공원에서 학생 대표들이 선언문 낭독 → 독립 만세 운동이 퍼져나가며 다양한 계층의 사람들 참여

● 일제의 탄압 : 합천 학살 사건(3월 16일), 제암리 학살사건(4월 15일)

● 의의 : 해외에서의 독립운동으로 이어짐, 일제의 지배방식이 변화하는 계기, 대한민국 임시정부 수립, 독립운동의 대중적 기반확대, 항일무장투쟁 활성화, 세계각국의 약소민족 운동에 영향

● 주요 인물 : 유관순, 권동진, 나용환, 최린, 한용운

★ 국외 운동 ★

1. 서간도 지역

● 경학사 : 1911년 설립, 항일독립운동단체 농업개발, 군사교육, 민족의식 고취

● 신흥무관학교 : 1911년 설립, 독립군 양성 학교, 이회영, 이시영, 청산리 전투 참여

● 부민단 : 1912년 설립, 이주 한인들 보호, 애국청년 육성, 독립운동 기초 마련

● 서로군정서 : 1919년 만주에서 조직된 무장독립운동단체, 독립군 양성, 일제통치기관 파괴, 민족반역자와 친일파 처단

2. 북간도 지역

● 중광단(대종교) : 1911년 조직된 항일독립운동 단체, 청년정신교육

● 대한국민회 : 3.1 운동 직후 결성된 항일독립운동 단체

● 서전서숙 : 1906년 설립, 한국최초 신학문 민족교육기관

● 명동학교 : 1908년 설립, 민족교육기관

● 북로군정서 : 무장독립운동단체, 김좌진

● 대한독립군 : 홍범도 장군, 봉오동전투

3. 러시아 연해주

● 성명회 : 블라디보스토크에서 조직, 독립운동단체

● 권업회 : 신한촌에서 조직된 항일 독립운동단체

● 대한광복군정부 -> 대한국민의회 : 망명 정부, 임시정부 성격
