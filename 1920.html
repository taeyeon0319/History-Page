1920년대 운동

-일제의 문화통치-

  • ※ 정치 : 문관 총독 임명, 보통 경찰제, 민족 신문 발행 허용
  • ※ 교육 : 신교육령(1922)
  • ※ 통치 : 치안유지법 제정(1925)
  • ※ 경제수탈 : 산미증식계획(1920~1934), 회사령 철폐(1920)

★ 국내 운동 ★

1. 대중 운동

● 학생 : 6.10 만세운동(학생단체, 사회주의 단체 계획 수립), 광주학생 항일 운동 (신간회의 지원, 기회주의 부인)

● 소년 : 방정환(천도교, 어린이날)

● 여성 : 근우회(신간회 자매단체, 좌우합작)

● 백정 : 형평 운동(진주)

2. 실력양성운동

● 물산장려운동 : 국산품 애용운동, 사회주의 세력이 비판함, 결과적으로 물가상승

● 민립대 설립운동 : 결과적으로 일본의 방해로 실패

3. 의열투쟁

● 조선혁명선언(신채호, 폭력혁명)

● 김상옥 : 종로경찰서 폭탄 투척

● 나석주 : 동양척식 주식회사 폭탄 투척

4. 문맹퇴치 운동

● 야학 설립

5. 신간회(1927)

● 비타협적 민족주의 세력 + 사회주의 세력 -> 민족유일당(좌우합작)

● 공개단체, 합법적 단체, 전국적 조직

● 광주학생 항일운동 지원

★ 국외 운동 ★

1. 봉오동 전투(홍범도, 대한독립군) -> 훈춘사건 조작

2. 청산리 전투 : 김좌진, 북로군정서

3. 간도참변

4. 밀산부 : 서일, 대한독립군단 조직

5. 자유시참변(러시아)

6. 3부 결성 : 참의부, 정의부, 신민부

7. 미쓰야 협정

8. 3부 통합운동 : 북만주-혁신의회, 남만주 국민부
